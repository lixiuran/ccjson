# JSON 在线解析工具

一个功能完整、界面美观的 JSON 在线解析工具，参考 [json.cn](https://www.json.cn/) 设计。

## 功能特性

- ✅ **JSON 格式化** - 自动美化 JSON 数据，提高可读性
- ✅ **JSON 压缩** - 去除所有空格和换行，压缩 JSON 数据
- ✅ **JSON 验证** - 实时验证 JSON 语法，显示详细错误信息
- ✅ **JSON 转义/反转义** - 处理 JSON 字符串转义
- ✅ **实时统计** - 显示字符数和行数统计
- ✅ **一键复制** - 快速复制处理结果
- ✅ **示例数据** - 内置示例数据，快速体验功能
- ✅ **键盘快捷键** - 支持常用快捷键操作
- ✅ **响应式设计** - 完美适配桌面和移动设备

## 使用方法

1. 在左侧输入框中输入或粘贴 JSON 数据
2. 点击工具栏按钮执行相应操作：
   - **格式化** - 美化 JSON 格式
   - **压缩** - 压缩 JSON 数据
   - **转义** - 转义 JSON 字符串
   - **反转义** - 反转义 JSON 字符串
   - **验证** - 验证 JSON 语法
   - **清空** - 清空所有内容
   - **示例数据** - 加载示例 JSON
3. 结果会显示在右侧输出框中
4. 点击 **复制** 按钮复制结果到剪贴板

## 键盘快捷键

- `Ctrl/Cmd + Enter` - 格式化 JSON
- `Ctrl/Cmd + Shift + C` - 复制结果
- `Ctrl/Cmd + L` - 清空内容

## 特性说明

### JSON 格式化
自动将压缩的 JSON 转换为格式化的、易于阅读的格式，使用 2 个空格缩进。

### JSON 压缩
去除所有不必要的空格和换行，生成最紧凑的 JSON 格式，适合网络传输。

### JSON 验证
实时检查 JSON 语法的正确性，如果发现错误会显示详细的错误信息，包括错误位置和原因。

### JSON 转义
将 JSON 字符串中的特殊字符转换为转义序列，如换行符 `\n`、制表符 `\t` 等。

### JSON 反转义
将转义序列还原为原始字符。
